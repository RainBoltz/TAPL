# TAPL Telegram Bot

## Overview

This is the Telegram Bot of Anonymous Number Lendings.

## Prerequisites

### Installation

```bash
$ cd tapl-bot
$ pip3 install -r requirements.txt
```

### Check Configuration

- variable `telegram_bot_token` in file `tap-bot/main.py`

## Usage

### start telegram bot

```bash
$ cd tapl-bot
$ python3 main.py
```
